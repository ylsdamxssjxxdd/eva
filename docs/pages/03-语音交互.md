# 03 语音交互：开麦即聊

[返回总览](GitHubPages用户引导.md) · [上一页：补完](02-补完.md) · [下一页：视觉交互](04-视觉交互.md)

## 🎯 适用场景
- 口述提问、听写文本，减少键盘操作。
- 会议速记、灵感记录，录音后自动转文字再问模型。

## 📦 你会得到
- 一键录音→转写→提问的完整链路。
- 音频上传、Whisper 模型选择的小技巧。

## 🛠️ 前置准备
1. 在“增殖-声转文”选择 Whisper 模型（首次按 F2 会提示选择）。
2. 确认麦克风可用，或准备 WAV/MP3/OGG/FLAC 文件。

## 🚀 操作步骤
1. 按 **F2** 选择/确认 Whisper 模型。
2. 再按 **F2** 开始录音，第三次按 **F2** 结束，自动转写并填回输入框。
3. 检查转写文本，补充上下文后，点击 **“发送”** 或 `Ctrl+Enter`。
4. 若已有音频文件，直接拖拽/上传；发送前会自动封装为 `input_audio`。

## 🧠 小技巧
- 追求速度：选小模型；追求准确：选大模型。
- 环境嘈杂时，在输入框补充关键词，提升识别稳定性。
- 长录音建议分段录制，逐条提问更高效。

## ❓ 常见问题
- 没声/没文本：检查麦克风权限，确认状态栏显示“录音中”。
- 乱码：优先 16k/16bit 单声道 WAV，或切换 Whisper 模型。
- 回复过长：在提问中限制字数或要点数。
