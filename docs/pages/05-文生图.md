# 05 文生图：幻想成真

[返回总览](GitHubPages用户引导.md) · [上一页：视觉交互](04-视觉交互.md) · [下一页：知识库](06-知识库.md)

## 🎯 适用场景
- 生成插画、封面、角色设定、概念草图。
- 需要从文字描述快速得到视觉方案。

## 📦 你会得到
- 在“增殖-文生图”完成一次生成的流程。
- Prompt 书写、风格模板和分辨率选择的小技巧。

## 🛠️ 前置准备
1. 打开“增殖-文生图”，确认扩散模型路径（默认 `text2image/sd1.5-anything-3-q8_0.gguf`）。
2. 选择分辨率、步数、CFG、风格模板（按需）。

## 🚀 操作步骤
1. 输入提示词（如“暖色科幻驾驶舱，全身角色”），可补充反向提示词。
2. 设定分辨率/步数/CFG，点击 **“生成”**。
3. 生成完成后，输出区显示图片标记与路径；可再次调整提示词继续生成。

## 🧠 小技巧
- 让对话区先帮写精简 prompt，再粘贴过来生成。
- 显存紧张：先用 512 边长测试，再提高分辨率。
- 想要特定风格/镜头/光线，直接写在提示词里。

## ❓ 常见问题
- 生成失败：查看输出错误片段，检查模型路径/显存；尝试降分辨率或步数。
- 图片路径太长：输出区已换行，可直接复制路径查看。
- 风格不准：补充风格关键词，如“赛博朋克”“水彩”“电影感逆光”。
